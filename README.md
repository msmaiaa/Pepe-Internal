# Pepe-Internal
Csgo intern4l cheat made with imgui. Not pasted but it's not a code to learn from, it has a lot of bad practices.  
Features as of 16/10/2021:  
* Visuals
  * ESP (boxes)
  * Glow
  * Radar
  * RCS Crosshair
  * Aimbot Fov
  * Show KDA
  * Show Ping
* Triggerbot
  * Toggled
  * Delay
* Aimbot
  * Toggled
  * Fov
  * Smooth aimbot?
  * Smooth aimbot speed
* Misc
  * Bhop
  * RCS
  * No Flash
  * Fov changer
  * Fov amount
* Config
  * Load config
  * Save config
## How to compile
Just have c++ and directx sdk stuff installed and you're good to go
